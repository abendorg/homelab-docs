# Step 8: InfinityFree for Basic Website Hosting

InfinityFree provides a free, simple hosting environment for public-facing experiments.

## What You’re Doing
You will configure a basic external web host for testing and sandboxing.

## Why InfinityFree
- Free  
- Easy to set up  
- Useful for public status pages or landing pages  
- Keeps your home network isolated  

## Actions

### 1. Sign Up
Use your new custom email address.  
Enable MFA if available.

### 2. Delete the Default Site
Start with a clean slate.

### 3. Add Your Custom Domain
Choose the option to point your domain via **A or CNAME records**, not nameservers.

### 4. Add DNS Records in Cloudflare
Follow InfinityFree’s instructions.

### 5. Security Warning
This is shared hosting.  
Do **not** store anything sensitive here.

## Use Cases
- Public status page  
- Simple landing page  
- Web experiments
